Variables

We all need to access pieces of data quick, and Trigger variables allow you to do just that. To see a variable's value, just type it into the second slot:

ee : server_path

You can also use Trigger variables like regular EE variables in the third slot. For example:

ee : templates : set base ({ee_path}templates)

In the above example, {ee_path} gets replaced by the full path to the your EE folder.
