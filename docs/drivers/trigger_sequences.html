Trigger Sequences

Driver slug: seq

The site driver deals with Trigger sequences.

run (sequence) or (package/sequence)

Runs a Trigger sequence.

ee : seq : run (pre_launch)
ee : seq : run (blog/build_blog)

list

Lists all the Trigger sequences in the sequences folder as well as in packages.
