Global Variables

Driver slug: globals

The global variables driver deals with EE global variables.

list

Lists all global variables.

new (name, [value])

Creates a new global variable. Optionally takes a value.

set (name, value)

Sets the value of a global variable.

delete (name)

Deletes a global variables.

delete all

Delete all global variables.
