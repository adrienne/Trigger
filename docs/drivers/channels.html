Channels

Driver slug: channels

The channels driver deals with EE channels.

new (channel title, [channel name], [channel description])

Creates a new channel. Trigger will create a channel name from the title, but you can provide one if you'd like as well.

list

Lists all channels.

enable versioning (channel name)

Enables versioning for a channel.

disable versioning (channel name)

Disables versioning for a channel.

enable format buttons (channel name)

Enables rich formatting buttons for a channel.

disable format buttons (channel name)

Disables rich formatting buttons for a channel.

enable comments (channel name)

Enables comments for a channel.

disable comments (channel name)

Disables comments for a channel.

enable captcha (channel name)

Enables CAPTCHA on commenting for a channel.

disable captcha (channel name)

Disables CAPTCHA on commenting for a channel.

set xml lang (channel name, language)

Sets the XML language for a channel. Can take the full country name of the two character code.

set default status (channel name, status)

Sets the default status value for a channel.

select allow comments by default (channel name)

Selects 'Allow Comments' button in Publish page by default for a channel.

deselect allow comments by default (channel name)

Deselects 'Allow Comments' button in Publish page by default for a channel.

allow image urls (channel name)

Allows image URLs in channel entries for a channel.

disallow image urls (channel name)

Disallows image URLs in channel entries for a channel.

enable auto links (channel name)

Enables automatically turning URLs and email addresses into links for a channel.

disable auto links (channel name)

Disables automatically turning URLs and email addresses into links for a channel.

set max revs (channel name, revs)

Set maximum number of recent revisions per entry.

clear rev data (channel name)

Clear all existing revision data for a channel.

set default title (channel name, title)

Set the default entry title for a channel.

set url prefix (channel name, prefix)

Set the URL prefix for a channel.
