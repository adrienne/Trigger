Site

Driver slug: site

The site driver deals with basic site operations and configurations.

site offline

Turns the site offline.

site online

Turns the site offline.

enable op

Enables the output profiler.

disable op

Disables the output profiler.

enable template debug

Enables template debugging.

disable template debug

Disables template debugging.

set debug (debug level)

Sets the debugging level. Allowed values are 0, 1, 2.

clear cache

Clears all caches.

clear page cache

Clears the page cache.

clear db cache

Clears the database cache.

clear rel cache

Clears the relationship cache.

set site name (site name)

Sets the site name.

set license (license number)

Sets the license number.

set index (index file)

Sets the site index. For blank, set it to 'blank':

ee : site : set index (blank)

set root (root url)

Sets the site root url. Can be used with the auto_base_url variable for easy setting.

ee : site : set root ({auto_base_url}/)

set cp index (cp index)

Sets the site control panel index. Can be used with the auto_base_url variable for easy setting.

ee : site : set root ({auto_base_url}/{system_folder}/index.php)

set theme url (theme url)

Sets the site theme url. Can be used with the auto_base_url variable for easy setting.

ee : site : set theme url ({auto_base_url}/themes)

set theme path (theme path)

Sets the site theme url. Can be used with the auto_base_url variable for easy setting.

ee : site : set theme path ({server_root}/themes)
