Variable Reference

server_root

The server path to the EE install.

server_name

The server name from PHP's SERVER_NAME.

license

The license number for the current EE install.

debug_level

Current debug level

site_name

Current site name.

app_path

Path to 'expresionengine' folder.

base_url

Returns the base url in the config (without a trailing slash).

auto_base_url

Generates and returns the base url based on the current environment (without a trailing slash).

system_folder

Returns the current system folder name.

path_third

Returns the PATH_THIRD constant. No trailing slash.
