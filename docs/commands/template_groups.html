Template Groups

Driver slug: groups

The template groups driver deals with EE template groups.

new (group_name, is_site_default)

Creates a group and the index template. Passing a 'y' to the second parameter will make that template group the default or index template group.

delete (group_name)

Deletes a template group and the templates within it.

The delete function is currently not functioning correctly.

list

Lists current template groups.

delete all

Deletes all groups and their templates.
