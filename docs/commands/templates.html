Templates

Driver slug: templates

The templates driver deals with EE templates (and sometimes groups).

sync

Syncs all templates. File templates and groups will be updated and created in the database. Things found in the database not found in the filesystem will be created.

new (group/template, content, [type], [save_as_file-y or n])

Creates a template within a group. If the group doesn't exist, Trigger will create it. Obeys the config that allows templates to be saved as files (if it is set to yes the necessary files will be created in the filesystem).

Additionally, if no group is supplied, it will be added to the site's default group.

enable strict urls

Enables strict URLs.

disable strict urls

Disables strict URLs.

save template revs

Sets save template revisions to yes.

dont save template revs

Sets save template revisions to no.

max number of revs (# of revs)

Sets the max number of template revisions.

allow as files

Allows templates to be saved as files.

disallow as files

Disallows templates to be saved as files.

set base (basepath)

Sets the template file basepath.
