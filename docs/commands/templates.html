Templates

Driver slug: templates

The templates driver deals with EE templates (and sometimes groups).

sync

Syncs all templates. File templates and groups will be updated and created in the database. Things found in the database not found in the filesystem will be created.

new (group/template, content, [type], [save_as_file])

Creates a template within a group. If the group doesn't exist, Trigger will create it. The type value will set the type of template to create. Set to webpage as default. Available types are:

  • webpage
  • static
  • feed
  • css
  • feed
  • xml

Setting the last value to 'y' will force the template's config to marked it as savable as a file. Leaving this blank will set it to your site's config.

delete (group/template)

Deletes a template. If a template file exists, it will delete this as well.

enable strict urls

Enables strict URLs.

disable strict urls

Disables strict URLs.

save template revs

Sets save template revisions to yes.

dont save template revs

Sets save template revisions to no.

max number of revs (# of revs)

Sets the max number of template revisions.

allow as files

Allows templates to be saved as files.

disallow as files

Disallows templates to be saved as files.

set base (basepath)

Sets the template file basepath.
