Templates

Driver slug: templates

The templates driver deals with EE templates (and sometimes groups).

sync

Syncs all templates. File templates and groups will be updated and created in the database. Things found in the database not found in the filesystem will be created.

new (group/template, content, [type], [save_as_file])

Creates a template within a group. If the group doesn't exist, Trigger will create it. The type value will set the type of template to create. Set to webpage as default. Available types are:

  • webpage
  • static
  • feed
  • css
  • feed
  • xml

Setting the last value to 'y' will force the template's config to marked it as savable as a file. Leaving this blank will set it to your site's config.

delete (group/template)

Deletes a template. If a template file exists, it will delete this as well.

enable strict urls

Enables strict URLs.

disable strict urls

Disables strict URLs.

save template revs

Sets save template revisions to yes.

dont save template revs

Sets save template revisions to no.

max number of revs (# of revs)

Sets the max number of template revisions.

allow as files

Allows templates to be saved as files.

disallow as files

Disallows templates to be saved as files.

set base (basepath)

Sets the template file basepath.

set 404 (group/template)

Sets the site 404 template. The template and group should already exist for this function.

enable cache (group/template, refresh interval)

Enables cache for a template with a refresh interval.

disable cache (group/template)

Disables the cache for a template and sets the refresh interval back to 0.

set refresh (group/template, refresh interval)

Sets the refresh interval for a template.

allow php (group/template, [parsing stage])

Allows PHP for a template. Parsing stage defaults to output, but this can be overridden by providing 'i' or 'input' for the second parameter.

disallow php (group/template)

Turns off PHP allowing for a template.

set php stage (group/template, [parsing stage])

Sets PHP parsing stage for a template. Parsing stage defaults to output, but this can be overridden by providing 'i' or 'input' for the second parameter.

change type (group/template, new type)

Changes the type of a template (see list of valid types above). If you are saving templates as files, this function will also rename the file with the proper extension.

rename (group/template, new template name)

Renames a template. If you are saving templates as files, this function will also rename the file.
